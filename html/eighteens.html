eighteens module

class eighteens.Eighteens(option: int)

Bases: Bet

A class used to represent a bet for eighteens. Children of Bet.

Parameters:

  • weight (int) – 1

  • options (list) – [1, 2]

  • option (int) – An acceptable bet option (1 for 1-18 or 2 for 19-36)

is_in_bet(number: int) → bool

Takes a number and checks if it is in the range of the bet (in the chosen option of it).

Parameters:

number (int) – The number that was drawn in the roulette.

Returns:

A bool which answers the title question (True if the number is in the range of the bet, False if it is not)

Return type:

bool
