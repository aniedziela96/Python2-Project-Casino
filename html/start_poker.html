class start_poker.Poker_Game(player: Player)

Bases: object

Class responsible for playing the poker game.

Parameters:

player (Player) – Player that will be playing the game

action() → None | str

Allows player to choose a action and plays it.

Returns:

string representing chosen action or None if player wants to check their tokens

Return type:

NoneType or str

static end_game()

Input that allows player to end the game

final() → None

Shows the player croupier’s hand and determines the winner and adds tokens to the player account

round_one() → None

Plays round one of poker, shuffles the deck, croupier and player draw two cards

round_two() → None

Plays round two of poker game, player and croupier draw three cards

start_game() → None

Plays the whole poker game.
