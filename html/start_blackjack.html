start_blackjack module

class start_blackjack.Strat_blackjack(player: Player, bet_money: int)

Bases: object

Class that is responsble for playing a blackjack game.

Parameters:

  • player (Player) – Player that will be playing blackjack

  • bet_money (int) – Tokens that player can win

action(bet_number=1, first=False) → str

Prints all possible options for player, and returns a chosen option, if this is first action for current bet, player can choose insurance (if the visible card of croupier is ACE) and split (if player has a pair in his hand)

Parameters:

  • bet_number (int) – number of bet that is played

  • first (bool) – True if this is a first action for bet number bet_number, False otherwise

Returns:

string corresponding to chosen option

Return type:

str

deal() → None

Deals two cards that are drawn from the deck to the croupier and player, then adds a new bet to the player’s list of bets.

decision(game: Blackjack, if_first=False) → None

After player chooses the move, function does action corresponding with chosen option and checks if the move is possible (if player has enough tokens to play).

Parameters:

  • game (Blackjack) – Blackjack bet that is played

  • if_first (bool) – True if this is the first decision in bet

final() → None

Checks the result of a game by comparing player’s scores with croupier’s score and ads tokens to player in case of a win or a draw.

game_status(hidden=True) → None

Shows a game, croupier’s hand and all of the player’s bets.

Parameters:

hidden (bool) – If hidden is True first croupier’s card isn’t shown.

play_bet(game: Blackjack) → None

Shows the current status of a game and lets a player choose an action

Parameters:

game (Blackjack) – Game that will be played

players_score() → list

Calculates the scores of all player’s bets

Returns:

list of scores

Return type:

list

start_game() → None

Starts the game of Blackjack, deals cards and iterates through the list of bets and playes each of them.
