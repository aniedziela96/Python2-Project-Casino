poker_game module

class poker_game.Poker(player: Player)

Bases: object

Class contaning methods that handle possible poker actions

Parameters:

player (Player) – Player playing poker

bet(player=True, all_in=False) → None

Plays the action of bet in poker game

Parameters:

  • player (bool) – If True player plays the bet, if False Croupier is playing Croupier move - under construction

  • all_in (bool) – True if player wants to bet all his tokens

fold(player=True)

Plays the fold

Parameters:

player (bool) – If True nothing happens - allows the game to end, if False player wins all the tokens because Croupier is automatically loses

match(player=True) → None

Plays the match action in poker game

Parameters:

player (bool) – If True player plays the bet, if False Croupier is playing Croupier move - under construction

winner() → Poker_human | Croupier

Establishes a winner of the poker game. First checks the rank of player and croupier, than compares it. If both ranks are the same uses a winner class to determine the winner

Returns:

player or croupier, depending on who one the poker game

Return type:

Poker_human or Croupier
