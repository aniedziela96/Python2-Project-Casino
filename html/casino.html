casino module

class casino.Casino(players_file: str)

Bases: object

A class used to represent a casino. Allows to create a new player or log in, to play casino games and to log out.

Parameters:

file (str) – A name of a file with players

check_tokens(tokens: int, player: Player, blackjack=False) → bool
Checks if player has enough tokens to spend and prints the message with

the information how many tokens the player needs.

Parameters:

  • player (class:player.Player) – Player who is spending tokens

  • tokens (int) – Amount of tokens

  • blackjack (bool, optional) – If True the is not being printed, defaults to False

Returns:

True if player has the right amount of tokens, False otherwise

Return type:

bool

create_player(name: str, tokens: int) → Player

Creates player based on a name and tokens.

Parameters:

  • name (str) – Player’s name

  • tokens (int) – Number of tokens

login() → Player
Allows to login, when player is logging for the first time creates new

account in players file.

Returns:

Player that has successfully logged

Return type:

Player

logout(player: Player) → None

When leaving the casino saves a current amount of tokens in players file.

Parameters:

player (Player) – Player who is logging out.

start() → None

Allows to enter the casino and choose a game a player wants to play. Initializes the chosen game.

casino.get_pass()

Creates a window that allows user to type the password without showing letters.

Returns:

A password that the player has typed.
