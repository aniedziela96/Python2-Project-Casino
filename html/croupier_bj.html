croupier_bj module

class croupier_bj.Croupier_bj

Bases: object

Class representing a blackjack croupier

croupiers_move(deck: Deck) → None

Plays a croupier move, draws cards from the deck and calculates score until croupier’s score reaches 17, if a score is greater than 21 prints a message that the croupier busted and changes score to 0

Parameters:

deck (Deck) – Deck from wich croupier will draw cards

draw_cards(cards: list) → None

Adds cards to croupier’s hand.

Parameters:

cards (list) – List of cards that will be added to croupier’s hand

set_score() → None

Calculates and sets croupier’s hand score, according to blackjack rules

show_hidden() → None

Prints a representation of croupiers hand, when first card is hidden

show_open() → None

Shows croupier’s cards
