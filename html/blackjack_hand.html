blackjack_hand module

class blackjack_hand.Blackjack_hand

Bases: Hand

Represents a hand in blackjack game.

calculate_score() → None

Calculates and sets the score for hand. Sums the value of each card in hand, ace has a value 11 or 1, depending what is best for player.

card_split() → tuple

Removes the second card from the hand.

Raises:

IndexError – This is only possible if there are exacly two cards in hand

Returns:

Card that has been removed

Return type:

tuple

is_pair() → bool

Checks if cards in hand have the same value

Raises:

IndexError – This is only possible if there are exacly two cards in hand

Returns:

True if both cards hae the same value, False otherwise

Return type:

bool
